启动

新建的集群需要首先格式化namenode

  1. 创建Zookeeper命名空间
    ./bin/hdfs zkfc -formatZK
  2. 启动journalnode服务
    在所有的journalnode服务器上启动
    ./sbin/hadoop-daemon.sh journalnode
  3. 在主Namenode节点格式化
    ./bin/hadoop namenode -format
  4. 在主节点启动namenode
    ./sbin/hadoop-daemon.sh start namenode
  5. 将主namenode节点格式化的目录拷贝到从主namenode节点上
    ./bin/hdfs namenode –bootstrapStandby
    ./sbin/hadoop-daemon.sh start namenode
  6. 在两个namenode节点都执行以下命令
    ./sbin/hadoop-daemon.sh start zkfc
  7. 在所有datanode节点都执行以下命令启动datanode
    ./sbin/hadoop-daemon.sh start datanode

  8. 在主namenode节点启动yarn，运行yarn-start.sh命令

    在各节点通过jps查看启动状况
